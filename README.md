Sonarc lets video editors search their local audio libraries 
using natural language. Instead of scrubbing through hundreds 
of tracks manually, just type "intense drum drop" or "soft 
ambient fade" and Sonarc returns the exact file and timestamp 
where that moment occurs.

Built with CLAP (Contrastive Language-Audio Pretraining), 
ChromaDB, FastAPI, and React + Tauri. Runs entirely offline 
on your machine — no cloud, no subscription, no data leaves 
your computer.
